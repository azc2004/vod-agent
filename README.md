# 🎬 Vod-Agent (하프클럽 상품 동영상 자동 생성기)

Vod-Agent는 하프클럽(Halfclub) 쇼핑몰의 상품 데이터를 수집하여, AI 비디오 모델과 이미지 처리 기술을 통해 고품질의 숏폼 홍보 영상을 자동으로 생성해 주는 Streamlit 기반 애플리케이션입니다.

정적인 상품 이미지에 생명을 불어넣어 **모델 워킹/회전 비디오**와 **상세 재질 컷**이 매끄럽게 연결되는 약 15초 분량의 매력적인 홍보 영상을 생성할 수 있습니다.

---

## 🌟 주요 기능 (Key Features)

1. **하프클럽 상품 데이터 수집**
   - 하프클럽 상품 고유 번호(Product ID)를 기반으로 대표 이미지, 추가 이미지 및 상세 설명 HTML 데이터를 실시간으로 파싱합니다.
   
2. **AI 모델 워킹 비디오 생성 (Google Veo 3.1 / Luma)**
   - **Google Veo 3.1** 및 **Luma Dream Machine** API를 활용하여 정적인 대표 이미지를 360도 회전하며 의류의 핏과 스타일을 보여주는 자연스러운 실사 패션 모델 비디오로 변환합니다.

3. **OpenAI GPT-4o-mini 비전 분석 및 스마트 크롭**
   - 이미지 내에 패션 모델(혹은 마네킹)이 존재하는지 분석하여 적합한 이미지인지를 자동으로 필터링합니다.
   - 상세페이지의 텍스트 배너, 공지, 사이즈 차트 등 불필요한 영역을 자동으로 인식하여 제외하고, **모델과 의류 위주의 깨끗한 영역만 크롭**합니다.

4. **MoviePy 기반 비디오 합성 & 모션 그래픽 연출**
   - 크롭된 디테일 컷 이미지들에 서서히 줌인(Zoom-in) / 줌아웃(Zoom-out)되는 모션을 적용해 원단의 재질과 질감을 입체적으로 노출합니다.
   - 클립 간 부드러운 전환을 위해 **크로스페이드(CrossFade)** 트랜지션을 적용하고, 고화질/저용량의 `.webm` 파일(VP9 코덱)로 인코딩하여 출력합니다.

5. **인터랙티브 웹 대시보드 (Streamlit)**
   - 최근 등록된 40개의 상품 리스트를 그리드 형태로 시각화합니다.
   - 이미 비디오 생성이 완료된 상품은 대시보드 내에서 즉시 플레이어로 감상할 수 있습니다.
   - 상품 번호를 직접 입력하거나 리스트에서 선택해 단 클릭 몇 번으로 동영상을 생성하고 다운로드할 수 있습니다.

## 📂 프로젝트 구조 (Project Structure)

```text
vod-agent/
├── app.py                  # Streamlit 웹 대시보드 및 사용자 인터페이스
├── video_generator.py      # 데이터 수집, 이미지 필터링/크롭, AI 연동 및 비디오 렌더링 로직
├── requirements.txt        # 프로젝트 의존성 라이브러리 목록
└── .streamlit/
    └── secrets.toml        # API 키 정보 저장 (로컬 보안 파일)
```

## 🚀 시작하기 (Getting Started)

### 1. 요구 사항
- Python 3.8 이상
- FFmpeg (비디오 인코딩용 라이브러리, 시스템 환경에 설치되어 있어야 합니다.)
  - macOS: `brew install ffmpeg`
  - Windows: [FFmpeg 공식 홈페이지](https://ffmpeg.org/download.html)에서 다운로드 후 환경 변수 등록

### 2. 가상환경 설정 및 패키지 설치
프로젝트 루트 디렉토리에서 아래 명령어를 실행하여 의존성 패키지를 설치합니다.

```bash
# 가상환경 활성화 (각 OS에 맞게 실행)
# macOS/Linux:
source venv/bin/activate
# Windows:
# .\venv\Scripts\activate

# 패키지 설치
pip install -r requirements.txt
```

### 3. API 키 설정
비디오 생성 및 이미지 스마트 크롭을 위해 Gemini 및 OpenAI API 키 설정이 필요합니다.
프로젝트 루트에 `.streamlit/secrets.toml` 파일을 생성하거나 수정하여 아래와 같이 키를 추가해 주세요.

```toml
# .streamlit/secrets.toml
GEMINI_API_KEY = "YOUR_GEMINI_API_KEY"
OPENAI_API_KEY = "YOUR_OPENAI_API_KEY"
```

## 🖥️ 사용 방법 (Usage)

### 1. 웹 대시보드로 실행
```bash
streamlit run app.py
```
- 브라우저에 대시보드가 열리면, 최근 상품 목록 중 하나를 **선택**하거나 우측 사이드바에 **상품 번호를 직접 입력**합니다.
- `🤖 AI 모델 워킹/회전 비디오 생성 활성화` 체크박스를 켜면 Google Veo 3.1 API를 호출해 인트로 영상을 생성합니다. (비활성화 시 정적 이미지 슬라이드쇼로 대체됩니다.)
- **동영상 생성 시작** 버튼을 누르면 데이터 수집부터 최종 렌더링까지 자동으로 진행되며, 완성된 영상을 웹 브라우저에서 바로 재생 및 다운로드할 수 있습니다.

### 2. CLI 명령어로 실행
웹 대시보드 없이 터미널에서 특정 상품 번호로 즉시 동영상을 렌더링할 수도 있습니다.
*(이 경우 환경변수 혹은 코드 내 설정이 필요합니다)*

```bash
python video_generator.py <상품번호>
# 예시: python video_generator.py 1234567
```

## ⚠️ 주의사항

- **API 비용 주의**: AI 비디오 생성(Veo 3.1) 및 GPT-4o-mini 이미지 분석 기능을 활성화하고 대량의 작업을 수행할 경우 API 호출 비용이 발생할 수 있습니다.
- **캐싱 제공**: 동일한 상품 번호에 대해 이미 생성된 비디오가 로컬에 존재할 경우 (`result_<상품번호>.webm`), 중복 API 호출 및 렌더링 없이 즉시 기존 결과물을 재사용합니다.
